# Tarjeta de Interprete(`<List />`)

## Descripción General

The `List` component is a reusable UI element designed to display an item, such as an actor or a movie character,
with an image, a name, and a description. It is accessible and supports highlighting items associated with highly-rated movies.

## Accesibilidad

- The component uses semantic HTML (`article`, `figure`, `header`).
- It includes `aria-label` to provide context to screen readers, especially when `esNota10` is true.
- The image has an `alt` attribute describing the content.
- It is focusable via `tabIndex="0"`.
